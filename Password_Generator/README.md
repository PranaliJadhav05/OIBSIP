# Simple Password Generator

This is a simple Python project that generates a random password.

## Description
The program asks the user for the password length and generates
a random password using letters and numbers.

## Features
- User chooses password length
- Generates random password
- Easy to use
- Beginner friendly

## How to Run
1. Make sure Python is installed
2. Open Command Prompt
3. Go to the project folder
4. Run this command:

python password_generator.py
